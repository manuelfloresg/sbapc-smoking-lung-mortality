# Support-Window Interpretation Notes

This diagnostic uses 200 common complete simulation seed(s) available in both the observed-window and full-support result directories.

1. Full-support SBAPC is compared with observed-window SBAPC as an oracle-style diagnostic, not as a feasible empirical estimator.
2. On average, the full-support estimator changes annual MARE by 1.0 percentage points relative to observed-window SBAPC, where positive values mean lower error under full support.
3. By horizon region, the observed-window MARE minus full-support MARE is: Credible: 1.1 percentage points; Caution: 1.0 percentage points; Risky: 0.9 percentage points; Full horizon: 0.9 percentage points.
4. The comparison should be interpreted as evidence about support truncation only. It does not isolate every remaining source of discrepancy from Truth, including smoothing, APC extrapolation, and the sequential mortality mapping.
5. Sex-specific differences: Sex-specific support-window summaries were not included in the compact table.
